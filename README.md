# PetFeeder
## Description: 
-------------------
The framework is based on IOTForce stack:
* Features in Salesforce:
    * Remote feeding
    * Scheduled feeding
* IOT Proxy:
    * Ioticos(MQTT)
*  Hardware:
    * ESP32
    * 5v stepper motor
* Software:         
    * Arduino IDE
    * Salesforce:
         * Custom Object: Plant
